# AI-Customer-Support-Log-Analyzer
AI-Customer-Support-Log-Analyzer

## Overview
This project analyzes customer support tickets using Machine Learning and Business Analytics.

## Features

- Dashboard with KPIs
- Ticket Status Analysis
- Ticket Type Analysis
- Priority Analysis
- Channel Analysis
- Product Analysis
- Customer Satisfaction Analysis
- Monthly Ticket Trends
- AI Ticket Category Prediction using NLP

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- TF-IDF
- Logistic Regression

## Machine Learning

Models Evaluated:
- Logistic Regression
- Multinomial Naive Bayes
- Linear SVM
- Random Forest Classifier

Final Model Deployed:
- Random Forest Classifier

Vectorization:
- TF-IDF

Deployment:
- Streamlit Community Cloud
